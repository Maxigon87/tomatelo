# tomatelo
Tomatelo es una aplicación simple y amigable que te ayuda a mantenerte hidratado durante el día. Registra fácilmente cada vaso de agua que tomas, visualiza tu progreso diario y sigue tus hábitos con estadísticas semanales. Con un diseño moderno y recordatorios inteligentes, Tomatelo te motiva a crear una rutina saludable de hidratación.
