# 🍅 Tomatelo

**Tomatelo** es una aplicación móvil simple y amigable que te ayuda a **mantenerte hidratado durante el día**.
Registra fácilmente cada vaso de agua que tomas, visualiza tu progreso diario y crea un hábito saludable de hidratación.

Diseñada con **Flutter + Material 3**, Tomatelo busca ofrecer una experiencia **minimalista, moderna y motivadora**.

---

## 💧 Características

✨ Registro rápido de vasos de agua
📊 Seguimiento del progreso diario
📈 Estadísticas semanales de hidratación
🎨 Interfaz moderna con **Material 3**
🌙 Modo oscuro elegante
🍅 Diseño amigable y minimalista
⚡ Animaciones suaves para una mejor experiencia

## 🚀 Instalación

1️⃣ Clona el repositorio

```bash
git clone https://github.com/tuusuario/tomatelo.git
```

2️⃣ Entra en la carpeta

```bash
cd tomatelo
```

3️⃣ Instala dependencias

```bash
flutter pub get
```

4️⃣ Ejecuta la app

```bash
flutter run
```
